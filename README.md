# Turnip Price Calculator: Animal Crossing New Horizons
Predict turnip prices and corner the stalk market in Animal Crossing New Horizons

See it in action at [https://artem6.github.io/acnh_turnips/](https://artem6.github.io/acnh_turnips/)


### `npm start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm run test`

Launches the test runner in the interactive watch mode.<br />

### `npm run build`

Builds the app for production to the `docs` folder for deploying to GH Pages.<br />
